Color

Color plays a major part in how a website looks and feels. CSS provides the developer with different ways to add color to their websites. These ways are:

  • Named Colors - turquoise
  • Hex - #57e36a
  • RGB - rgb(150, 220, 97)
  • HSL - hsl(300, 60%, 74%)

The RGB and HSL methods for CSS color also have another variation that contains an a - rgba and hsla. The a value controls the opacity of the color.

Text Styles

The way that different text is displayed can make a website more organized. There are multiple headers and paragraph html tags that a user can use to present text. These styles have different default font weights and font sizes.

H1: Main Page Heading

  • Font: Mukta
  • Font Weight: Bold
  • Font Size: 35px

H2: Page Subheading

  • Font: Mukta
  • Font Weight: Bold
  • Font Size: 25px

P: Page Paragraph Text

  • Font: Oxygen
  • Font Weight: 16px
  • Font Size: Normal

Position

The position CSS property allows the developer to arrange the many elements on a webpage in specific ways. Elements can be shifted, put in a specific spot, or fixed on the page.

Relative

The relative attribute shifts the block element a certain amount relative to its original position.

Absolute

The absolute attribute removes the element from the flow of the website and places it in a designated spot

Fixed

The fixed attribute fixes an element to a specific spot on the webpage screen. The element will stay in the same spot even while scrolling.
